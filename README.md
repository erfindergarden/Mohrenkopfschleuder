
# Marshmallow Slingshot
pc: Mohrenkopfschleuder, Schokokussschleuder

First Version from Minirevollo for the MAKER FAIR ROMA 2017. ([original Github](https://github.com/minirevollo/Mohrenkopfschleuder))


***Please use the newest Inkscape version 0.92, because the older Version used 90 dpi and the newest version 96 dpi to use the same dpi as webbrowser like chrome. A 1 inch object saved in an older version of inkscape would be 6% smaller (90 Pixel / 96 Pixel per inch = 0.9375 inch) in inkscape 0.92.***

***If you use Illustrator to cut your slingshot use the pdf Version to avoid scaling. Illustrator uses 72 dpi. A object would be 33% bigger (96 Pixel/ 72 Pixel per inch = 1.33 inch) then in inkscape.***

Version V6

For Maker Faire Berlin Mini improved the design a bit. He made the following improvements. 
* Sperrmuttern an allen beweglichen Teilen
* Verbesserte Standfestigkeit durch 3 mm extra Breite
* Verschraubung der senkrechten Querstrebe
* Schraubenschlüsseldesign mit optimierter Hebelwirkung

The stop nuts are a bit difficult to screw on so if it is your first sling shot you should start with the v5 Version.

## Design

Drawn with Google SketchUp. Exportiert as 2D model in Format DXF.

<img src="IMG/Mohrnkopfschleuder_v_4_bild_3.jpg" width = "30%" /> <img src="IMG/Mohrnkopfschleuder_v_4_bild_2.jpg" width = "30%" /> <img src="IMG/Mohrnkopfschleuder_v_4_bild_1.jpg" width = "30%" />

Weiterbearbeitet in Inkscape. Hier wurden noch die Gravuren auf einem extra Layer und in einer anderen Farbe angelegt. Schriften mussten dann noch in Pfade umgewandelt werden.

Beim Import der DXF Datei aus Sketchup entstehen keine durchgehende Pfade der einzelnen Objekte. Dadurch fährt der Laser die einzelnen Pfade der Objekte Stück für Stück ab. Er schneidet nicht durchgehend. Um das zu vermeiden müssen alle Pfade eines Bauteils ausgewählt werden. Danach die Pfade kombinieren. Dann alle Pfade mit dem Knotenwerkzeug + Strg-Taste auswählen. "Gewählte Knoten vereinigen" drücken. Jetzt ist jedes Bauteil ein durchgehender Pfad.
